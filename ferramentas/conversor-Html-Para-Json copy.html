🧠 Text → JSON Converter Inteligente

Cole qualquer texto e deixe a ferramenta detectar o formato e transformar em JSON válido.

Como funciona

A ferramenta tenta múltiplos parsers em cascata e escolhe o melhor resultado. Você também pode forçar um modo específico.

AutoDetecta automaticamente: JSON, CSV, TSV, YAML simples, INI, key:value, listas, XML/HTML, tabelas markdown ou texto livre.
CSV / TSVPrimeira linha vira as chaves. Suporta aspas, vírgulas escapadas e delimitador automático.
Key:ValueLinhas do tipo nome: João ou idade = 30. Blocos separados por linha em branco viram array de objetos.
ListaCada linha (ou item com -, *, 1.) vira um elemento do array.
Markdown TableConverte tabelas | col | col | em array de objetos.
XML / HTMLEstrutura de tags convertida em árvore JSON com atributos e filhos.
YAML simplesIndentação com chave: valor e listas - item.
Texto LivreExtrai sentenças, parágrafos, emails, URLs, números e datas em um objeto rico.
💡 Dica: use Auto na maioria dos casos. Se o resultado não for o esperado, escolha o modo específico e clique em Converter.
